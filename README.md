#### 安装教程

1.  运行sql
2. 打成jar包或者直接在idea运行
3.  需要开启redis
4.  需要开启ES环境 我使用的版本为7.6.1 然后修改

#### 使用说明

技术
1. ssm
2. spring boot版本2.2
3. maven 3.6.X
4. java 1.8
5. 博客搜索使用Es7.6.1 需要搭建ES环境  linux环境建议使用docker win直接运行ES即可
6. 关于博客浏览量  简单的使用redis  设置  重复刷新浏览量不增加
